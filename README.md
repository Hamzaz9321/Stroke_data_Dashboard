# Stroke_data_Dashboard
This Streamlit-based dashboard provides a comprehensive visualization of a healthcare dataset related to stroke. The dashboard includes a variety of interactive charts and graphs, enabling users to explore and analyze the data effectively.

Features:
1. Data Preview
Displays the first few rows of the dataset to provide an overview of the data.
2. Scatter Plot
Visualize relationships between two continuous variables.
Options to select different columns for the X and Y axes.
3. Histogram
Explore the distribution of a single variable.
Select a column to generate a histogram with adjustable bin size.
4. Pie Chart
Display the proportion of categorical variables.
Choose from various categorical columns to generate the pie chart.
5. Bar Chart
Compare values between different categories.
Select columns for both the X and Y axes to generate a bar chart.
6. Line Chart
Analyze trends over a continuous variable.
Options to select columns for the X and Y axes to create a line chart.
7. Tree Map
Hierarchical visualization of data.
Choose hierarchical columns and a value column to create a tree map.
8. Heatmap & Confusion Matrices
Correlation Heatmap: Visualize the correlation matrix of numerical variables.
Confusion Matrices: Display confusion matrices for different models (KNN, SVM, RF).
